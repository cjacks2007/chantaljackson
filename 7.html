Customer Churn Prediction with Machine Learning

This project predicts customer churn using a Telco customer dataset. It applies logistic regression to identify patterns in customer behavior that correlate with churn, helping companies proactively address customer retention

Key Features

  1. Simulated dataset of 1,000 customers (tenure, services, contract type, billing)
  2. Full ML pipeline: preprocessing, scaling, modeling, evaluation
  3. Confusion matrix and classification report to assess model accuracy
  • Insight: The initial model struggled with class imbalance (few “Yes” churns predicted). In a real-world scenario, I would apply techniques like SMOTE or try tree-based classifiers.
  • Tools: Python, pandas, scikit-learn, seaborn, matplotlib
